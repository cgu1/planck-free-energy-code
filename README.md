# Code for "Planck Law from a Classical Free Energy Extremum"

This repository contains the code used to generate the figures and numerical
results in the paper:

Carlos Gomez-Uribe, *Planck Law from a Classical Free Energy Extremum*,
Quantum Studies: Mathematics and Foundations (2025).

## Contents
- `simulation.py`: main simulation code
- `analysis.ipynb`: analysis and plotting

## Requirements
- Python 3.9+
- numpy
- scipy
- matplotlib

## Usage
Run:
```bash
python simulation.py
